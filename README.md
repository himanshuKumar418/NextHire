# NextHire - Placement Preparation Platform

NextHire is a MERN-stack placement preparation portal built to support final-year placement prep. It tracks topic completion, company-specific preparation paths, and provides practice materials.

## Project Structure

```text
nexthire/
├── client/                 # React.js Frontend
└── server/                 # Node.js/Express.js Backend
```

## Tech Stack
*   **Frontend:** React.js, React Router DOM, Tailwind CSS, Axios
*   **Backend:** Node.js, Express.js
*   **Database:** MongoDB, Mongoose
*   **Auth:** JWT & bcrypt

## Features
*   Interactive Dashboard
*   Placement Tracker (Countdown Timer)
*   Company Readiness Score
*   Smart Revision Lists
*   Practice Hub (DSA, Aptitude, Core Subjects)
